# Customer Churn Analysis & Prediction Project

## Overview
This project is an end-to-end data analytics internship project covering data cleaning, exploratory data analysis (EDA), machine learning, and dashboard development.

## Objectives
- Clean and preprocess raw datasets
- Perform exploratory data analysis
- Build predictive models (Regression & Classification)
- Perform clustering analysis
- Develop interactive dashboards using Power BI

---

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)
- Google Colab
- Power BI
- GitHub

---

## Project Structure

### Level 1: Data Cleaning & EDA
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (Iris Dataset)

### Level 2: Machine Learning
- Regression Analysis
- Clustering (K-Means)

### Level 3: Advanced Analytics
- Classification (Churn Prediction)
- Power BI Dashboard Development

---

## Key Insights

- Churn rate stands at 14.6%
- Most churn occurs among low-engagement customers (0–1 support calls)
- Mid-tenure customers are at higher risk of churn
- Higher product usage is associated with customer retention
- Feature adoption improves retention rates
